A Simple Wordpress Plugin That Deletes All Updating of Plugins, Themes, and Even The Wordpress Core.



== Description ==


This plugin simple removes the updating of the plugins, theme, and even the wordpress core.
This way you can pick your favourite version of the plugin (theme, or wordpress core) and you won't get update notices.

= <a href="http://wordpress.org/plugins/disable-updates/">Disable Updates</a> Downloaders =
This plugin is the updated version of that plugin.



= Please note!!! =


It is very important to keep your WordPress plugins up to date. 
If you use this plugin, make sure your plugins are not harming your website.

== Installation ==


1. Go to the Plugins> Add New> Upload in you admin dashboard.

2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.

== Frequently Asked Questions ==


Q: <strong>Are there any settings?</strong>
A: No, their may be in later updates.

Q: <strong>If I download this, and then I deside to deavtivate it will Wordpress allow updating again?</strong>
A: Yes, Wordpress will allow updating again.
